# Two-Stage CMOS Op-Amp Schematic Design

## Overview

This project presents the schematic design and simulation of a two-stage CMOS operational amplifier using Cadence Virtuoso.

## Tools Used

- Cadence Virtuoso
- Spectre Simulator

## Design Specifications

- Technology: 180 nm CMOS
- Supply Voltage: 1.8 V
- Open Loop Gain: 55 dB (Pre-layout)
- Phase Margin: 60 degree

## Schematic Description

The operational amplifier consists of:

- Differential input stage
- Current mirror load
- Second gain stage
- Miller compensation capacitor
- Biasing circuit
